# LINE風会話ログビルダー

HTMLの会話ログから、LINE風のトーク画面を生成・編集・出力するWebアプリです。  
ブラウザだけで動作し、生成した画面は単体HTMLとして保存できます。

## 主な機能
- HTML会話ログの読み込み（ドラッグ&ドロップ対応）
- LINE風トーク画面の自動生成
- 編集モードで編集・挿入・並び替え
- 画像メッセージの追加とサイズ調整
- 単体HTMLとして出力

## 動作環境
- Google Chrome / Microsoft Edge などの最新ブラウザ

## 使い方
1. `index.html` をブラウザで開きます。
2. 会話ログHTMLをドラッグ&ドロップして「生成」を押します。
3. 編集モードをONにして、編集・挿入・並び替えを行います。
4. 「作成」を押して単体HTMLをダウンロードします。

## 編集モードについて
- **編集**: 各メッセージの「編集」ボタンから内容を変更  
- **挿入**: そのメッセージの後ろに新規メッセージを追加  
- **並び替え**: メッセージをドラッグして上下に移動  

## 画像メッセージのサイズ調整
編集モード中、画像の右下ハンドルをドラッグするとサイズ変更できます。  
縦横比は維持されます。

## 出力形式
「作成」で生成されるのは、画像を埋め込んだ単体HTMLです。  
そのまま配布・共有できます。

## GitHub Pagesでの公開（簡易）
1. GitHubで新規リポジトリを作成  
2. このプロジェクトのファイルをリポジトリにアップロード  
3. リポジトリの **Settings → Pages** を開く  
4. **Branch: main / (root)** を選んで保存  
5. 数分後に表示されるURLへアクセス
